物料模型关联工单 使用说明

发表于 2019-04-22 | 分类于 工单管理 | | 阅读次数: |
字数统计: 1k

在执行代维维护任务时，可能会遇到发现问题，需要更换/修补设备的情况。
所以要使用不同类型的物料进行维护任务，这时记录每个任务使用的具体物料信息就尤为重要，物料模型功能也由此开发上线。

由于使用物料环节是代维人员操作，所以增加的物料功能均在代维人员回单环节



¶通用任务工单

¶登录APP

登录掌上运维大厅 -> 更新 代维工单(新) APP -> 进入更新后的APP，选择通用任务工单

¶添加物料信息

  1. 点击工单，填写完成情况，并拍照确认现场情况

  1. 点击 是否需要使用物料 选项，决定是否使用物料。 点击 是 时，下方会出现 添加物料 的信息。点击 添加物料，选择 任务类型、使用场景后，直接点击 查询 按钮，即可查询出 属于该任务类型、场景的物料信息。

  1. 选择完物料信息后，点击右上角的 √ 符号，进入到物料信息详情页面。在这个页面进行 填写物料的具体使用情况、拍照、删除等操作 （ 注意：物料信息详情页面的物料信息，在未提交之前，保存后是可以进行修改删除的 ）

¶通用工单-演示视频

¶传输按次计费工单

¶登录APP

登录掌上运维大厅 -> 更新 传输计费工单 APP -> 进入更新后的APP，选择传输计费工单

¶添加物料信息

  1. 选择某个任务类型的菜单，点击工单信息，点击 回复按钮，进行回单操作。

  1. 点击 是否需要使用物料 选项，决定是否使用物料。 点击 是 时，下方会出现 添加物料 的信息。点击 添加物料，选择 任务类型、使用场景后，直接点击 查询 按钮，即可查询出 属于该任务类型、场景的物料信息。

  1. 选择完物料信息后，点击右上角的 添加 按钮，进入到物料信息详情页面。在这个页面进行 填写物料的具体使用情况、拍照、删除等操作 （ 注意：物料信息详情页面的物料信息，在未提交之前，保存后是可以进行修改删除的； 物料信息只能在 回复环节填写，阶段回复不支持此操作 ）

¶传输计费工单-演示视频

¶干线按次计费工单

¶工作量回复

登录网络代维系统，在 工单管理 菜单中选择 干线按次工单 ，点击 待办工单 ，选择工单号，进入回单页面。填写工作量情况

¶填写物料信息

选择任务类型、业务场景，在是否使用物料中，选择 是 。然后根据实际情况选择物料信息，并上传现场使用物料的照片信息。


以上就是物料模型使用手册的全部内容，如果您对这篇文章有任何问题请及时联系我，我的Email：machiyuan@boco.com.cn
