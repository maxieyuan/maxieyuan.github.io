动态巡检使用说明

发表于 2018-11-22 | 分类于 巡检管理 ， 动态巡检 | | 阅读次数: |
字数统计: 1.8k

动态巡检，根据每月巡检记录、告警信息、集团规范，动态的生成下一个月的计划。

包括的专业有：基站、传输线路、集客

¶动态巡检计划审批

每月根据七项运维系统生成的健康度得分，由网路代维系统进行生成计划信息。

各个地市分公司对需要巡检的计划进行查看、审批下发，由一线代维人员对下发的计划中的任务进行巡检，保障各个设备的正常运行。

由于三类专业的审批流程都大体一致，所以这里只以线路的作为演示

¶关联巡检资源

关联时可以选择在页面点击操作一栏的是/否来决定该资源的巡检，还可以点击导出按钮，对Excel表格进行操作，来批量修改。修改后点击从Excel导入修改即可。

¶计划确认

点击计划详情页面的计划确认按钮，即可下发计划。

**注意：**计划下发后，不允许回退至审批环节。确认时要注意！

¶线路巡检管理

基站与集客的动态巡检信息，均在工单管理中巡检工单功能中进行查看，之前也讲解过如何使用巡检工单。

这里这次只针对传输线路的线路巡检管理功能进行介绍

¶线路资源维护组划分

¶资源数据来源
  • 数据来源：管线系统
  • 维护厂家：国讯
  • 同步周期：1.线路段：天粒度；2.敷设点：按需同步
  • 不同步以下几类资源：集客A级光缆、集客AA级光缆、集客AAA级光缆、集客家客普通用户光缆、引入光缆、用户光缆

注意：如需要同步敷设点等大量数据时，请派EOMS工单，并保证每月月底之前需要同步的数据已录入至管线系统。

¶干线划分维护组

由于干线有特殊的需求，存在线路段分别位于两个区县之间，需要由两个区县的代维小组进行巡检。

所以干线资源可以划分两个及两个以下的班组，方便巡检。

  • 对于单条或者少数几条线路段划分班组时，可以勾选资源名称前的复选框，勾选之后，点击关联所选项，选择要关联的班组即可。

  • 对于多条或者较多条数线路段划分班组时，可以先根据特定的条件查询出资源，再点击页面下方的导出所有项或者导出所选项，将资源导出至Excel表格中。对表格中的资源进行批量的修改设置其班组信息，修改完成后，在页面最下方的从Excel导入功能，将表格再导入至系统即可。
  • 使用Excel导入功能时，如果要关联多个班组，则需要在代维公司一列填写班组时，使用,号进行分割班组即可
¶本地网划分维护组

本地网划分维护组时，只能一个资源对应一个班组，具体操作与干线划分维护组一致，不在赘述

¶计划制定

计划制定是之前动态巡检计划还未正式上线时，由各个地市代维公司进行人工手动选择资源，进行下发计划的一个功能。

由于现在已经上线了动态巡检，很多人已经不再使用该功能。

这里再一次的介绍该功能，是为了以后可能区公司会下发新的要求，需要使用该功能进行执行。

¶新建巡检计划

  1. 填写计划名称：尽量按照规范填写，例如：中移铁通呼和浩特分公司-赛罕区-11月手动派发巡检计划

    尽量要与动态巡检计划能够区分开，并且容易识别。

  2. 代维专业：根据实际情况进行选择，一般只分为：干线、本地网

  3. 所属地市：根据实际计划下发的地市进行选择

  4. 所属区县：根据实际计划下发的区县进行选择

  5. 巡检年：创建计划的当前年

  6. 巡检月：创建计划的当前月，或者下一个月

  7. 计划描述：非必填项

创建完计划后，可以选择哪些资源作为巡检项，选择完之后，直接提交审核。

审核人一般为地市移动管理员

¶我创建的计划

在新建巡检计划之后，可以在我创建的计划中看到我们新建的计划信息，以及当前计划的状态。

计划状态一般分为：

  1. 待审核
  2. 已通过

当我们查看计划的状态变更为已通过的时候，就可以在手机和PC端看到该计划以及计划下的巡检任务

¶待审批列表

这时我们切换账号，使用地市移动管理员的账号，在他的账号下，我们可以在待审批的计划中看到之前提交审批的计划

点击计划信息后的审批按钮，开始对计划进行审批操作：通过 or 驳回

审批通过后，可以在线路巡检统计的线路巡检完成情况统计中查找到该计划

¶巡检线路信息查询

主要功能用于对线路段的敷设点、GIS查看资源进行查看，对难度系数进行调整

¶GIS查看

查看某一个代维人员特定时间范围内的巡检记录。一般作为判断该巡检人员是否正常巡检的依据。

  1. 小组名称：非必填
  2. 巡检人：填写巡检人的中文姓名
  3. 联系电话：非必填
  4. 巡检日期：选择开始和结束时间，尽量保证在3-4天之内，避免数据过多，查询过慢

¶线路巡检统计

分两个功能，一个是完成情况统计，统计各个计划完成情况；另一个为全区统计，按月统计，统计十二个地市的完成率、覆盖率

¶线路巡检完成情况统计
  1. 计划名称：填写计划的大概名称即可，系统会自动模糊匹配
  2. 巡检维护组：选择某一个区县的维护组，即可查询到属于该维护组下的计划信息
  3. 年份：默认为当前年
  4. 月份：默认为当前月

当选择和填写了以上信息，即可点击统计按钮，查询出符合条件的计划信息了

¶线路巡检统计
  1. 选择年份：默认为当前年
  2. 选择月份：默认为当前月
  3. 专业类型：默认为所有专业，可以根据需求进行选择

统计表中的计划公里数占总公里数的比率就是集团定义的覆盖率，干线要达到100%，本地网要达到75%以上

巡检率则是集团定义的完成率，要达到100%


以上就是动态巡检的使用说明
